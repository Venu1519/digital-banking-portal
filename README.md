# Digital Banking Portal

A secure banking web application built using Python, Flask, PostgreSQL, HTML, CSS, and Bootstrap.

## Features

* User Registration & Login
* Deposit Money
* Withdraw Money
* Fund Transfer
* Transaction History
* PDF Bank Statement Download
* Admin Dashboard
* PostgreSQL Database
* Render Cloud Deployment

## Tech Stack

* Python
* Flask
* PostgreSQL
* HTML/CSS
* Bootstrap
* Git & GitHub
* Render
